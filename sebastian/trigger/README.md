# trigger

> 上级索引：[sebastian/](../README.md)

## 模块职责

负责主动触发引擎，使 Sebastian 能够在无用户输入的情况下自主发起任务（定时任务、事件监听、条件触发等）。当前处于 **Phase 4 占位阶段**，`__init__.py` 为空模块，尚未实现任何触发逻辑。

## 目录结构

```
trigger/
└── __init__.py    # 占位入口，当前为空（Phase 4 未实现）
```

## Phase 4 规划方向

主动触发引擎预计包含以下能力：
- **定时触发**：cron 表达式或间隔任务（如每日摘要、定时提醒）
- **事件触发**：订阅 Event Bus，响应特定系统事件（如文件变更、消息到达）
- **条件触发**：监控指定条件，满足时自动发起 Task
