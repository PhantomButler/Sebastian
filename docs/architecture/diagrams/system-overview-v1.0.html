② Agent Loop 详细流程

一次完整 turn 从用户消息进入到 SSE 推流的全链路。 双向 generator（BaseAgent ↔ AgentLoop）是核心机制：ToolCallReady 向上 yield，工具结果通过 asend() 注入回去。

③ 三层 Agent 委派链路

Sebastian（depth=1）通过 delegate_to_agent 异步委派给组长（depth=2）， 组长通过 spawn_sub_agent 派生组员（depth=3）。所有层均复用同一 BaseAgent 引擎。
